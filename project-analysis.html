Hidden Features

隐藏功能

通过编译时 Feature Gate 控制的高级功能，部分仅限内部使用。

🐧

BUDDY — AI 宠物伴侣

Feature Gate: feature('BUDDY')  |  路径: src/buddy/
基于账户 UUID + 盐值 (friend-2026-401) 的 FNV-1a 哈希确定性孵化。18 个物种 (鸭/鹅/猫/龙/章鱼/猫头鹰/企鹅...)，5 级稀有度 (普通 60% / 非凡 25% / 稀有 10% / 史诗 4% / 传说 1%)，加上独立的 1% 闪光概率。500ms ASCII 精灵动画。
18 物种 5 稀有度 1% 闪光 FNV-1a 哈希 /buddy pet /buddy hatch /buddy card
🧠

KAIROS — 持久化助手模式

Feature Gate: feature('KAIROS')  |  路径: src/assistant/
跨会话持久化，每日自动日志写入 ~/.claude/logs/YYYY/MM/YYYY-MM-DD.md。AutoDream 记忆整合 — 每 24h+ 且 5+ 新会话后触发，经历 Orient → Gather → Consolidate → Prune 四阶段。主动模式下 Claude 可自行触发工作。
跨会话持久化 AutoDream 四阶段 主动模式 SleepTool 15s 自动后台化
🚀

ULTRAPLAN — 云端深度规划

Feature Gate: feature('ULTRAPLAN')  |  路径: src/commands/ultraplan.tsx
将复杂问题卸载到云端 Opus 进行最长 30 分钟的独立深度研究。流程：本地 Prompt → 远程 CCR 会话 → Opus 独立研究 → 轮询结果 → 浏览器审查 → 本地执行。支持 Git Bundle 代码上下文传输。仅限内部使用。
30 分钟深度研究 远程 CCR Git Bundle 传输 仅限内部
🤖

Coordinator — 多智能体编排

Feature Gate: feature('COORDINATOR_MODE')  |  路径: src/coordinator/
Coordinator 仅持有 Agent/SendMessage/Shutdown 工具，将工作分派给拥有完整工具集的 Worker 子进程。核心规则：禁止模糊委托 — Coordinator 必须理解需求。共享任务列表存储于 ~/.claude/tasks/。
Coordinator-Worker 模型 子进程隔离 共享任务列表 禁止模糊委托
🌐

Bridge — 远程终端控制

Feature Gate: feature('BRIDGE_MODE')  |  路径: src/bridge/
WebSocket 双向连接本地 CLI 与 claude.ai 网页端。支持远程发送/接收消息、远程权限审批、实时输出查看与会话持久化。
WebSocket 双向 远程审批 实时输出流 33 文件
🎤

Voice — 语音交互

Feature Gate: feature('VOICE_MODE')  |  路径: src/voice/
Nova 3 语音识别引擎，支持流式 STT (Speech-to-Text) 与关键词检测。
Nova 3 STT 关键词检测 流式识别
